# Flaxen Tyranny Site

This is a static website for *The Flaxen Tyranny*.

## Purpose
A high-conversion landing site for showcasing the book, including:
- Video trailer
- Audiobook samples
- Character/worldbuilding content
- Conversion funnel to purchase

## Deployment
This site can be deployed via:
- GitHub Pages
- Netlify
- Vercel

## Structure
- index.html (main page)
- css/ (styles)
- js/ (scripts)
- img/ (images)
